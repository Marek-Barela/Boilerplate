# Boilerplate

This i my next.js, redux-saga, typescript boilerplate

## Instalation

Clone repository and run:

```bash
npm install
```

## MaterialUI

Boilerplate contain MaterialUI configuration just change the branch

```bash
git checkout MaterialUI
```
